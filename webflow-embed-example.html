Custom Google Map for Webflow

How to embed this map in Webflow:

  1. Add a div element with id="map" where you want the map to appear. You can use data attributes to customize the map, add markers, and set custom marker icons: 
    <div id="map" 
     data-lat="51.5074" 
     data-lng="-0.1278" 
     data-zoom="12"
     data-min-zoom="5"
     data-max-zoom="16" 
     data-api-key="YOUR_GOOGLE_MAPS_API_KEY"
     data-styles-url="https://cdn.jsdelivr.net/gh/hlabsdev1/cavandish@main/map.json"
     data-map-type="roadmap"
     data-show-controls="true"
     data-scroll-zoom="true"
     data-open-google-maps="false"
     data-marker-icon="https://cdn.prod.website-files.com/660ece57e4eaffd4b634b1d3/68554215b5e667ab1c603edc_Snazzy%20Maps%20Marker.png"
     data-marker-width="44"
     data-marker-height="63"
     data-marker-1="51.5074, -0.1278, London"
     data-marker-1-icon="https://cdn.prod.website-files.com/660ece57e4eaffd4b634b1d3/68554215b5e667ab1c603edc_Snazzy%20Maps%20Marker.png"
     data-marker-1-url="https://london.gov.uk"
     data-marker-2="51.51150025413154, -0.07672723460545847, 1 America Square"
     data-marker-3="51.500149, -0.126240, London Eye"
     data-marker-3-icon="https://cdn.prod.website-files.com/660ece57e4eaffd4b634b1d3/68554215b5e667ab1c603edc_Snazzy%20Maps%20Marker.png"
     style="height: 500px; width: 100%;"></div>
  2. In the Webflow page settings, add a custom code embed in the <head> section with this script: 
    <script src="https://cdn.jsdelivr.net/gh/hlabsdev1/cavandish@main/map-embed.js"></script>
  3. That's it! The map will automatically initialize with the settings from your data attributes.

Map Configuration Attributes:

  • data-api-key: Your Google Maps API key (optional, can be embedded in the script)
  • data-styles-url: URL to a JSON file with map styles (optional)
  • data-lat and data-lng: Center coordinates for the map
  • data-zoom: Zoom level (1-20, higher is more zoomed in)
  • data-min-zoom: Minimum zoom level (1-20, prevents zooming out too far)
  • data-max-zoom: Maximum zoom level (1-20, prevents zooming in too far)
  • data-map-type: Map type - "roadmap", "satellite", "hybrid", or "terrain"
  • data-show-controls: Show or hide map controls (true/false)
  • data-scroll-zoom: Allow zooming with mouse wheel (true/false)
  • data-open-google-maps: Open Google Maps when clicking on the map (true/false)

Marker Attributes:

  • data-marker-icon: Sets a default icon for all markers
  • data-marker-width: Sets a default width for all marker icons (in pixels)
  • data-marker-height: Sets a default height for all marker icons (in pixels)
  • data-marker-1, data-marker-2, etc.: Marker locations in format "lat, lng, title"
  • data-marker-1-icon: Sets a custom icon for marker 1
  • data-marker-1-width: Sets a custom width for marker 1 icon (in pixels)
  • data-marker-1-height: Sets a custom height for marker 1 icon (in pixels)
  • data-marker-1-url: URL to open when marker 1 is clicked

You can use Google's built-in marker icons or your own custom marker images.
